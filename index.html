Referencia de Uso

Agregar repositorio
echo "deb [trusted=yes] https://lexusytg.github.io/Nano-Machine/ stable main" | sudo tee /etc/apt/sources.list.d/nano-machine.list
Actualizar paquetes
sudo apt update && sudo apt upgrade
Instalar paquetes del repo
sudo apt install <paquete-nano>
Verificar repositorio
apt list --installed | grep nano-machine
